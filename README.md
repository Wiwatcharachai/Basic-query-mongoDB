## Simple MongoDB Data Generator

### Usage

- Clone repo นี้ลงไปบนเครื่อง
- ตรวจสอบว่า MongoDB เปิดอยู่
- ให้ `cd` เข้ามาที่ Repo นี้ แล้วพิมพ์ Command `node app.js` ลงไปใน CLI
